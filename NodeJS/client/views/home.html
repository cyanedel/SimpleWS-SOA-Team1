Home Page of simple web service project.

This web page is started in NodeJS. All requests called from this page is directed to NodeJS Backend. From there NodeJS will forwards requests to appropriate business services

Customer List page is connected to ws/api based on python with postgreSQL as database

Movie List page is connected to ws/api based on go(lang) with MongoDB as database

Animal List page is connected to ws/api based on PHP with MySQL as database

Database mockup generated from mockaroo (https://www.mockaroo.com/)
